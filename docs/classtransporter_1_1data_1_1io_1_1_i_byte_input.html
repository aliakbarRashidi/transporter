virtual transporter::data::BufferPtr transporter::data::io::IByteInput::readBytes ( std::size_t  count)
pure virtualnoexcept

Attempt to read count bytes from the underlying input and return them in a pointer to a newly-constructed transporter::data::Buffer.

The returned buffer is:

  • At most count bytes, less if not enough bytes were available (including 0)
  • nullptr if there was an error
