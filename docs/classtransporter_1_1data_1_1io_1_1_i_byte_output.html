virtual ssize_t transporter::data::io::IByteOutput::writeBytes ( const transporter::data::Buffer &  buffer)
pure virtualnoexcept

Attempt to write the bytes in buffer to the underlying output and return the number of bytes written.

The returned number of bytes written is:

  • At most the size of buffer, less if not everything could be written (including 0)
  • -1 if there was an error
Parameters
bufferReference to the buffer containing the bytes to write
Returns
Number of bytes written or -1 in case of error
