◆ readBytes()

transporter::data::BufferPtr transporter::network::io::NetworkStream::readBytes ( std::size_t  count)
overridevirtualnoexcept

Attempt to read count bytes from the underlying input and return them in a pointer to a newly-constructed transporter::data::Buffer.

The returned buffer is:

  • At most count bytes, less if not enough bytes were available (including 0)
  • nullptr if there was an error
Parameters
countNumber of bytes to read
Returns
Pointer to newly-constructed transporter::data::Buffer containing the read bytes

Implements transporter::data::io::IByteInput.

◆ writeBytes()

ssize_t transporter::network::io::NetworkStream::writeBytes ( const transporter::data::Buffer &  buffer)
overridevirtualnoexcept

Attempt to write the bytes in buffer to the underlying output and return the number of bytes written.

The returned number of bytes written is:

  • At most the size of buffer, less if not everything could be written (including 0)
  • -1 if there was an error
Parameters
bufferReference to the buffer containing the bytes to write
Returns
Number of bytes written or -1 in case of error
